Zewnętrzne arkusze stylów w CSS

1. Podstawowa składnia CSS

Przykład kodu CSS: 
body {
    background-color: #f0f0f0; 
    font-family: Arial, sans-serif;
}
p {
    color: blue; 
    font-size: 16px; 
}
h1 {
    color: green; 
    text-align: center;
}

2. Jak podłączyć arkusz CSS do HTML?

Arkusz stylów jest dołączany w sekcji <head> za pomocą znacznika <link>. Można dodać kilka plików CSS.

3. Podstawowa składnia podłączenia pliku CSS

4. Ile arkuszy CSS można dołączyć do jednej strony?

Nie ma ograniczeń co do liczby dołączonych arkuszy CSS, ale każdy plik powinien być umieszczony w oddzielnym znaczniku <link>.

5. Co się dzieje w przypadku konfliktów stylów?

Jeżeli ten sam styl zostanie zdefiniowany w różnych plikach, obowiązuje zasada kaskadowości – ostatni wczytany plik ma pierwszeństwo.

6. Tworzenie arkusza stylów w edytorach

  • Edytor tekstowy: Możemy napisać kod CSS w notatniku i zapisać go jako plik .css.
  • Edytor wizualny: Narzędzia jak Visual Studio Code czy Sublime Text oferują funkcje podglądu i ułatwień.

7. Jakie rozszerzenie ma plik CSS?

Każdy plik zawierający styl CSS zapisujemy z rozszerzeniem .css.

8. Zalety stosowania zewnętrznych arkuszy stylów

  1. Możliwość stosowania tych samych stylów na wielu stronach.
  2. Łatwe zarządzanie wyglądem – zmiany dokonane w jednym miejscu wpływają na cały serwis.

9. Dobre praktyki w CSS

  • Pliki .css należy podłączać w sekcji <head>.
  • Zaleca się stosowanie kilku arkuszy dla różnych sekcji strony, aby zachować porządek.
  • Konflikty stylów rozwiązuje kaskadowość – ostatni zapisany styl ma najwyższy priorytet.
