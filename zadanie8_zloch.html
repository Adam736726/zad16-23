Wewnętrzne style CSS

  1. Gdzie umieszczamy style wewnętrzne?

    Blok stylów znajduje się w sekcji <head> dokumentu HTML.

  2. Jak wygląda składnia wewnętrznych stylów?

  3. Rodzaje selektorów w CSS

    CSS pozwala na wybór elementów za pomocą:

    • nazw tagów (np. p, div)
    • identyfikatorów (#unikalne-id)
    • klas (.nazwa-klasy)
    • pseudo-klas (:hover, :focus)
    • atrybutów ([type="text"])

  4. Właściwości CSS

    Możemy modyfikować m.in.:

    • wygląd tekstu (czcionka, kolor, rozmiar)
    • marginesy i odstępy
    • obramowania i tła
    • układ elementów
